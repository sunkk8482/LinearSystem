# LinearSystemCarProject 자동차 화면 구현

## 개요
이 프로젝트는 3D STL 모델을 사용하여 자동차의 주행 시뮬레이션을 실행하는 MATLAB 스크립트입니다.

## 상황 설명
- **처음**: 자동차는 직진합니다.
- **교차로 앞**: 일시 정지합니다.
- **교차로에서 우회전**: 모든 자동차가 지나간 후 우회전합니다.
- **우회전 후**: 직진합니다.

## 구현된 화면 설명
### 첫번째 화면
- 한 지점에서 도로를 비추는 모습
- 좌표는 변하지 않습니다.
### 두번째 화면
- 현재 자동차를 따라가는 화면
- 자동차가 우회전을 하면 화면도 회전합니다.
### 세번째 화면
- 운전자 시점에서 화면이 비춰집니다.
